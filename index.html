KONFIGURASI SPREADSHEET

Dapatkan URL ini setelah deploy Apps Script sebagai Web App.

Default: tanggal hari ini. Ubah jika perlu input data untuk tanggal lain.

Cara Setup Google Apps Script
  1. Buka spreadsheet target ↗
  2. Klik menu Extensions → Apps Script
  3. Hapus kode default, paste isi file apps-script.gs
  4. Klik Deploy → New deployment
  5. Type: Web app — Execute as: Me — Access: Anyone
  6. Klik Deploy → izinkan akses → copy URL
  7. Paste URL di field di atas, klik Simpan

Setiap kali kode Apps Script diubah, buat New deployment baru.
